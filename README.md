## Features

- **Semantic HTML5**: Fully accessible and structured markup.
- **Custom CSS3**:
  - CSS Variables for easy theming (Light/Dark mode).
  - Flexbox & Grid layouts.
  - Smooth animations and transitions.
  - Responsive design (Mobile First).
- **Vanilla JavaScript**:
  - Dark Mode toggle with persistence.
  - Mobile navigation menu.
  - Smooth scrolling.
  - Form validation (Client-side).
- **Performance**: Zero external dependencies (No Bootstrap/Tailwind).

## Tech Stack

- **Languages**: HTML5, CSS3, JavaScript (ES6+)
- **Tools**: VS Code, Git
- **Design System**: Custom "Exotic Dark" theme with Neon accents.
